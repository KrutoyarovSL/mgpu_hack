Final Delivery

Customer Experience Platform

Финальная обертка над кейсом: FastAPI для inference-логики, Streamlit для demo и BI-ready слой для storytelling и дашбордов.

FastAPI

API-слой под churn scoring, retention recommendations и sales forecast.

uvicorn final_lending.api:app --reload
  • /predict_churn/{user_id}
  • /recommend/{user_id}
  • /forecast_sales

Streamlit Demo

Интерактивная оболочка для выбора клиента и просмотра результатов модели.

streamlit run final_lending/streamlit_app.py

BI / Storytelling

Используйте артефакты из final_lending/artifacts как слой выгрузки для DataLens, Superset, Power BI или Tableau.

Как запускать

  1. Подготовить данные: по умолчанию используются m_pred/archive/data.csv и m_pred/archive/events.csv.
  2. Запустить API.
  3. Запустить Streamlit.
  4. При необходимости задать переменные FINAL_LENDING_DATA_PATH, FINAL_LENDING_EVENTS_PATH, FINAL_LENDING_API_URL.
